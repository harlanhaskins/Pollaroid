# PollOption

A Poll object contains the information about a poll in our system.
It contains the following keys:

| Key | Type | Description |
| --- | ---- | ----------- |
| `id` | Int | The database identifier. |
| `text` | String | The textual contents of the option. |

Example: 

```json
{
  "id": 1,
  "text": "Abstain"
}
```
